# Blinkit Sales & Performance Dashboard (Power BI)

## Overview
This Power BI dashboard analyzes Blinkit sales, product categories,
and outlet performance to support data-driven decision-making.

## Tools Used
- Power BI
- Microsoft Excel
- Power Query

## Key KPIs
- Total Sales
- Average Sales
- Number of Items
- Average Rating
